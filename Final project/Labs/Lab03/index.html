Published on September 15, 2025 by Adelaja Oluwasimisola

My First Webpage

At first, making my first webpage felt confusing because there were so many tags and rules to follow. Once I learned how HTML works, it started to make sense, and I realized how much control I have over how a page looks. Creating this page helped me understand how websites are built step by step.

What I Learned

  1. How to use HTML tags to organize a page.
  2. How to make lists and headings look neat.
  3. How to add links that open in a new tab.

More About My Experience

  • HTML is easier than I expected once I practiced.
  • Writing code by hand helped me remember the tags.
  • I learned the importance of validating my code to fix errors.

Steps I Followed

  1. I created a folder called Lab03 in my CSIT121 folder.
  2. I made a new file named index.html and added HTML code.
  3. I tested my file in a browser to make sure it displayed correctly.
  4. I validated my code with the W3C validator and fixed the errors.

First Reflection

In Lab 02, I learned how to create a basic HTML page using headings, paragraphs, and links. I practiced structuring a webpage properly and began to understand how coding standards make content easier to read and maintain. This lab helped me build the foundation of web development.

Second Reflection

In Lab 03, I learned how CSS can transform the appearance of a webpage by separating content from design. Using selectors, colors, and styling made my pages look more organized and visually appealing. I also discovered the importance of using classes and IDs for layout, organization, and reusability.
