My First Webpage

At first, making my first webpage felt confusing because there were so many tags and rules to follow. Once i learned how HTML works, it started to make sense,and i realized how much control i have over how a page looks. Creating this page helped me understand how websites are built step by step.

What I Learned

  1. How to use HTML tags to organise a page.
  2. How to make lists and headings look neat.
  3. How to add links that open in a new tab.

More About My Experience

  • HTML is easier than I expected once i practiced.
  • Writing codes by hand helped me remember the tags..
  • i learned the importance of validating my code to fix errors.

Steps I Followed

  1. I created a folder called lab02 in my CSIT121 folder.
  2. I made a new file named index.html and added HTML code.
  3. I tested my file in a browser to make sure it displayed correctly.
  4. I validated my code with the W3C validator and fixed the errors.
  5. I uploaded my folder to GitHub and published it.

first reflecton
