Project Table — All 10 Labs

Below is a summary of the ten labs I completed during this course. Each lab built on the previous work and helped me master different parts of front-end development.

lab 01

In this lab, I downloaded and edited an HTML file, previewed it in a browser, checked it for errors, and then uploaded it to a web server so it could be viewed online. This taught me the basic steps of publishing a webpage.

View Lab 01

lab 02

In Lab 02, you practice writing HTML by hand using a text editor. You create a simple blog-style webpage that reflects what you’ve learned so far about web standards and HTML. The final page should look similar to the example provided, but with your own content.

View Lab 02

lab 03

In this lab, you add basic CSS to your Lab 02 blog page to improve its appearance. You create a stylesheet, link it to your HTML, and include another reflection about what you’ve learned.

View Lab 03

lab 04

In this lab, you build a multipage website that uses images and CSS to add visual elements. You include different image formats, apply background and aesthetic styles, and follow accessibility and design best practices.

View Lab 04

lab 05

In this lab, you update your Lab 03 blog page to make it responsive. You create a single-column layout for mobile devices, a multi-column layout for desktops using flexbox, grid, or media queries, and add link anchors for easy navigation within the page.

View Lab 05

lab 06

In this lab, you evaluate the usability of a public website using UI/UX principles you learned, including organization, navigation, CRAP design principles, and accessibility. You choose a non-eCommerce, informational website and assess how well it follows web design best practices.

View Lab 06

lab 07

In this lab, you build a two-page website: one page uses an HTML table to create a tier list, and the other page includes a form to collect user input. Both pages are styled with CSS for accessibility and consistency.

View Lab 07

lab 08

In this lab, you update your Lab 04 multipage website by adding an interactive navigation menu using HTML, CSS, and JavaScript. You choose a menu style, integrate it into your site, and customize it so it fits your design.

View Lab 08

lab 09

In this lab, you complete an XML tutorial, create an XML file with corrected and added data, and build an HTML page that reads and displays the XML content.

View Lab 09

lab 10

In this lab, I created a multipage website that teaches the class about the PNG web standard. I explained what PNG is, why it is used on the web, and its main features like lossless compression and transparency. The site uses proper structure tags, follows accessibility guidelines, and is designed using the CRAP principles.

View Lab 10

What I Learned

  • How to structure a multi-page site with semantic HTML
  • How to create responsive layouts using flexbox and media queries
  • How to build accessible forms, tables, and images
  • How to design using CRAP principles Constrast, Repetiton, Alignment and Proximity
  • How to validate code to avoid errors
  • How to organize files and folders for a web project

Lessons Learned Reflection

This project had a mix of challenges and successes for me. I struggled at times with getting my links, images, and CSS to work correctly, but fixing those issues helped me understand HTML and CSS much better. One of my biggest triumphs was seeing all my pages come together into a working website that I built on my own. Overall, this project taught me patience, problem-solving, and how to organize my files while creating a functional and clean website.

Observations

This project had both challenges and successes. I struggled with organizing my files and fixing display issues, but solving them helped me understand HTML and CSS better. Testing my website on different browsers and devices showed me how layouts can change, which taught me the importance of checking compatibility. Overall, building the site improved my problem-solving skills and gave me a clearer idea of how real websites are created and tested.
